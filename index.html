El Problema: Fragilidad y Exposición

Escenario Actual (Pymes)

Las infraestructuras locales (On-Premise) de las pequeñas empresas presentan vulnerabilidades críticas:

  • SPOF (Single Point of Failure): Un fallo de disco destruye años de operativa.
  • Ransomware: Aumento del 26% de secuestro de datos por exposición de redes locales.
  • Shadow IT: Los usuarios usan USBs inseguros ante la falta de herramientas corporativas.

La Solución Arquitectónica

Despliegue de un entorno Cloud externalizado centrado en tres pilares:

  • Alta Disponibilidad: Infraestructura AWS EC2.
  • Confidencialidad: Tráfico cifrado (SSL) y credenciales hasheadas (Bcrypt).
  • Integridad: Panel CRUD web aislado por sesiones lógicas de PHP.

Ciberseguridad: Defensa en Profundidad

La seguridad del servidor no depende de una única barrera, sino de múltiples capas concéntricas. Si una falla, la siguiente mitiga el impacto.

// Mitigación de Directory Traversal en PHP
$archivo_limpio = basename($_GET['file']);
// Bloquea inyecciones tipo "../../etc/passwd"

1. Seguridad Perimetral

UFW: Regla estricta Deny-All entrante. Solo apertura de puertos críticos TCP: 22 (SSH), 80 y 443 (Web).

2. Seguridad de Transporte

Certbot / Let's Encrypt: Cifrado SSL y redirección forzada. Protege las credenciales de ataques Man-in-the-Middle.

3. Seguridad de Aplicación (Blocklist)

Rechazo a nivel de código de ficheros .php o .sh para prevenir vulnerabilidades de Ejecución Remota (RCE).

Optimización de Recursos (Sostenibilidad)

Restricciones de Hardware

La instancia t3.micro del Free Tier de AWS cuenta con CPU limitada y solo 3.9 GB de almacenamiento libre real.

Decisión Técnica: Desplegar un demonio de copias masivas (Cron) era inviable por riesgo de colapso de disco. Se pivotó a un CRUD bajo demanda administrado por el usuario.

Almacenamiento JSON Ligero

Instalar el servicio MySQL habría consumido gran parte de la memoria RAM. Para gestionar usuarios se implementó un archivo JSON.

// Protección criptográfica BCRYPT
$hash = password_hash($pass, PASSWORD_BCRYPT);

Seguridad algorítmica sin el sobrecoste relacional.
